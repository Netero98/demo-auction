# Зачем вам этот форк ?

Данный форк добавляет к оригиналу лишь этот документ. Этот документ решает серьезный недостаток основного курса - он является лишь логом действий автора, что хорошо для объяснения, но не даёт понимания, как использовать актуальную версию проекта в случае, если вы захотите начать пилить, например, стартап на базе данного проекта (который весьма хорошо продуман для этой цели). Чтобы начать делать свой проект, вам придётся просмотреть весь курс и дойти до актуальной версии кодовой базы. Тут лежит выжимка из курса, которая поможет вам без просмотра более 100 часов основного материала сайта Deworker Pro уже сегодня запустить полноценный проект на основе самой актуальной версии кодовой базы. Информация дана в виде неупорядоченных топиков.

## Как поднять проект локально для разработки?
make init. Это главная команда при локальной разработке, суть её в том, чтобы полностью с нуля собрать и поднять проект, обновив всё что только можно и удалив перед этим всё старое включая базу данных. С этой команды надо начинать рабочий день. Для её выполнения нужен только установленный Docker на локальной машине.
